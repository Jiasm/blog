2025 年应该是工作这么久以来身心最为健康的一年了，因为合同明年就要到期，预感应该是不会被续约，所以主线目标就变成了保重身体，当你知道身边这些同事在最多一年之后就不会再见面，上班这件事情就变得轻松了许多，不会再给自己额外的压力、责任心，按时吃饭，按时下班，核心就是：差不多得了。

# 工作总结

25 年的 AIGC 是飞速发展的一年，国内外的各个厂家都在卷新的模型能力，配套的工程也一直在迭代，今年一年其实我已经很少 coding 了，大部分的代码都是在与模型的对话过程中完成的，今年一年的工作更多的是偏向设计（定义系统应该以什么样的流程运转）、审核（code review，结果验证），而具体的实施工作都是由模型完成的了。  
主要使用的模型是 Anthropic 家的 Claude 系列，今年应该是从 3.7 - 4 - 4.5 一路体验过来，借助 copilot 的体验权限对 Opus 和 Sonnet 两个版本都进行了尝试，到现在在使用的 4.5，个人感觉 Opus 和 Sonnet 的区别已经没有很大了，下半年可能因为是 GitHub 太久没有进行提交，所以我的 copilot 体验权限被取消了，后续转战了 Claude Code，发现随着模型能力的提升，我并不需要一个 **编辑器** ，工具的形态是丰富的，核心还是模型能力是否足够强力。  

也就是因为这个念头，下半年与团队一位伙伴合作了 remote agent 的开发，我认为随着模型能力的提升，AI Coding 应该逐渐从人与 AI 的结对编程演进为人向 AI 发布任务，等待验收，同时在企业内部的场景，如果能够打通企业内的基建工具平台，这将会是一个非常能够提升效率的解决方案，在工具测试版发布后我们进行了小规模的宣发，前来咨询的业务方也是蛮多的，说明我们的判断是正确的，但由于团队的主线目标和这个工具稍有出入，很遗憾这个方向没有持续做下去，算是有些可惜吧。  

在邻近年底时，又回到了去年底参与的生图方向上，开始打磨一款新的 agent 产品，随着 AIGC 一年的演进，大家对于 AI 的看法也都发生了很大的变化，借助新的模型，新的业界解决方案，希望我们能够在这个领域做出更好的成绩，对个人来说能够收获一份通用 agent 工程的建设经验也是非常棒的。  

今年一年我对于 coding 这件事本身的看法是非常纠结的，最初我是因为兴趣入门，但当它变成一种赚钱的手段后，便再没有通过 coding 这件事获得过成就感，希望退休后能够让 coding 回归兴趣爱好🙏尤其当 AIGC 这样发展，有想法变得更加重要，之前很多想法未能落地都是因为实现的成本还是有些高的，而现在只要是自己能够想明白一件事情，基本都可以靠模型折腾出来。

# 生活总结

我个人一直有记账的习惯，每到年底也是特别热衷于去各个 App 刷年度报告，所以针对家庭的收支一直都在做记录，中途也调整过很多次的统计方式，体验过一些 App，也自己尝试 vibe 过一个工具，最终还是选择白嫖了腾讯云文档，再不济还可以把文件下载下来以后转离线，现在能嫖就嫖，满足我随时记一笔、分享、做图的各种诉求。  
早年有过一段激进的记账，每周帮老婆统计她的开销，不过这个方式太容易给人压力了，所以后边就没再搞了，我就只是在统计自己的开销，今年年中我给她看了我的记账记录，以及每月开支的一些趋势图表，只是看到精美的图表和总结性的数字确实是很容易吸引到人，所以老婆说她也想要这样的记账记录，于是现在月底我会帮她也统计一份数据出来，算是对几年前没有进行下去的事情的一次升级吧。  
目前记录的范围包括我们约定的存款计划、计划内的各项开支抵扣，分类开销，实际攒下的钱，房贷的还款记录，通过记录这些数据能够比较清晰的定位问题，随着消费降级的大趋势，我们也借助这份数据砍了一些非必要的开支，在出去旅游时也可以根据存款计划的预算来规划行程安排，算是在看不清未来的环境里提供了一份可靠的指引吧。  

时间过得很快，我们搬进自己家也有三年多了，所以周边的外卖已经有些吃腻了，再加上我们定期去看消费账单，发现吃饭通勤几乎占据了大半的开销，为了避免厨房浪费，我们开始尝试周末自己做饭吃，因为我爸是烧伤科的医生，所以我从小对于明火明电都是比较惧怕的，去年岳父手把手的教过几次炒菜，算是克服了一些心理障碍，所以就开始了我们周末的做饭活动。  
先用自己曾经做过的番茄炒蛋，麻婆豆腐来建立自信，之后通过 **下厨房 App** 逐步扩展，最近已经成功完成过红烧牛肉这类硬菜了，幸好有小象超市，选购的肉类食材可以挑选片、块等切割好的，并不需要自己拿回家再来处理，给新手厨师降低了很多烹饪的门槛。  
现在周末已经很少会点外卖了，自己做的饭吃了一段时间后发现，确实是用料健康、实惠，因为有记账的缘故，很明显的可以看到在饮食分类下的支出在降低，虽说周末花在做饭上的时间可能有个三四小时，但最后吃上成品还是蛮有成就感的。  

今年出去玩的次数算是比较多的，过年时候在日本，劳动节去了长沙，国庆节去了泰国。  
对于日本大体印象没有特别深刻，因为和国内太像了，从文字到古建筑等，到处都可以看到国内的影子，但是一些细节确实还是比较触动人的，有一次坐了日本的公交车，发现它竟然会在靠站的时候把车门开启的那一侧车身压低，方便一些腿脚不方便的人上下车，日本餐馆店家都非常的客气，有次去吃一些海鲜，点了一只龙虾，在吃完身子之后我们觉得差不多了，准备走，老板过来一脸歉意，用蹩脚的英文告诉我们 wait、sorry，然后一边当着我们的面将龙虾腿压碎，示意我们里边是有肉的，一边道歉一边把所有的腿，钳子都给碾碎了放在盘子了。  
以及日本人好像是比较习惯用鞠躬来表达情绪，有次在地铁站发现有人帽子掉了，我示意之后，他连续三次鞠躬才走，还有前边提到的海鲜店老板，第二天路过她们店之后刚好看到了她，隔了老远还在给我们招手，鞠躬，单说在日本享受服务，确实是能够得到充足的情绪价值的。  

劳动节去长沙算是比较临时起意吧，主要目的是想考查一下长沙是不是适合作为养老的城市，因为长沙算是一线城市里平均房价最低的了，连吃几天湘菜，非常的对胃口，唯一费解的是餐馆大部分提供的都是热水，反观日本无论多冷提供的都是加冰块的水，这两处如果能调换就好了 🫠，所以有些理解茶颜悦色为什么会铺的大街小巷都是吧，毕竟这辣菜吃起来，热水真的喝不下去。  
因为是抱着考察养老的目的去的，所以在长沙的几天并没有去特别多的知名景点，去了很多河堤，湿地公园，还进了一次售楼处，单说房价、饮食，长沙确实是非常适合的了，但是作为一个南方没暖气的城市，冬天的体验会让人有些动摇，所以也只是将长沙放入备选清单中了。  

国庆选择泰国的原因就很简单了，高强度上班大半年，两人都很累了，需要找个地方躺平，是的，去泰国就是吃饭捏脚的，这次泰国旅程没有购买任何一张门票，目的地除了饭馆就是马杀鸡，吃饱喝足就在酒店的沙滩躺着晒太阳听海浪声，不过这次明显发现泰国的商业化已经过于严重了，消费水平也上去了，吃饭打车什么的成本比长沙还要高。  
当然也有一种原因是因为随着年龄增长，心里却越来越怂了，20年去曼谷住的小民宿，吃路边电线杆的小脏摊，这次我们特意去了之前住的地方和那个小脏摊，本想重温一下，但是看到串串上的苍蝇在飞，两人对了一下眼神，还是去吃别的馆子了。  

总的来说今年是非常快乐的一年，因为较好的实现了生活工作的隔离，今年自我感觉多了许多活人气息，而不再是工位 <-> 床的机器。

# 对未来的期许

我们很早之前就认为互联网人会在 35 岁时退休，不管是主动还是被动，互联网的强度显然不是适合持续更久的工作状态，身边有太多的朋友30来岁一堆的老年病，所以希望未来自己能够拥有健康的身体，不必为了这短暂的工作去透支自己，现在自己有非常多的待做事项，都受限于现在在上班没有时间去做，无论在快手的这份工作何时会迎来结束，我都要给自己放一个假，工作这么久以来每次换工作都没有给自己留出休息时间，一定要利用这个空档期来去做一些自己真正想做的事情。  
每当想到自己很快就可以去真正做一些有意义的事情，就非常的激动，也正是这个目标一直在支撑着自己在公司呆着，毕竟礼包额度是一直在膨胀的😂我也会认为这个事件会是一个非常重要的节点，闷头在企业做事情会让自己的视线变得非常狭窄，我也需要这样一个契机来审视自己想要的是什么。
